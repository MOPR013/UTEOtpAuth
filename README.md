# UTEOtpAuth

Ứng dụng Android (Kotlin) demo **đăng ký / đăng nhập** với **OTP qua email**.  
Có các màn: Sign In, Sign Up, Verify OTP, Forgot Password, Home.

## Tính năng
- Gửi & xác thực OTP qua email.
- Đăng ký tài khoản mới, kích hoạt bằng OTP.
- Quên mật khẩu bằng OTP.
- UI tối giản theo Material 3.

## Công nghệ
- **Kotlin**, **Android Studio** (Gradle Kotlin DSL)
- View/Jetpack (tuỳ theo dự án)
- Email/OTP provider (placeholder – cập nhật theo backend thực tế)

## Yêu cầu môi trường
- Android Studio Hedgehog/Koala trở lên
- JDK 17 (khuyến nghị)
- Android SDK 24+ (minSdk tùy project)
